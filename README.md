# AI Phishing Detection System

An AI-powered phishing email detection platform built using Flask and Scikit-learn.

## Features

- Phishing email classification
- TF-IDF NLP analysis
- URL and keyword detection
- Confidence scoring
- Flask web dashboard
- Machine learning based prediction

## Technologies Used

- Python
- Flask
- Scikit-learn
- Pandas
- NumPy
- HTML/CSS
- NLP

## Installation

```bash
pip install -r requirements.txt
python app/train.py
python app/web.py
